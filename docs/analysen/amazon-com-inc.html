  1. AWS-Wachstums-Reacceleration: 28 % in Q1 2026 ist das stärkste Wachstum seit 15 Quartalen — AI-Workloads (Bedrock + Trainium + Inferentia) treiben Nachfrage
  2. Custom Silicon (Trainium, Inferentia): Amazon baut eigene AI-Chips (Trainium 2, 3) → Kostenvorteil vs. NVIDIA-GPU-Miete; gleichzeitig Marvell-Kunde für Custom ASIC
  3. Anthropic-Investment: Amazon hat ~$4 Mrd. in Anthropic investiert (Claude API auf AWS Bedrock) — strategische AI-Partnerschaft mit $16,8 Mrd. Investitionsgewinnen in Q1 2026
  4. Advertising-Engine: $17,2 Mrd. (+24 %) — Amazon Ads ist die profitabelste Business-Einheit nach AWS und wächst schneller als Meta/Alphabet-Werbung

Marktposition: AWS ist der größte Cloud-Infrastruktur-Anbieter der Welt mit 30 % Marktanteil, dem tiefsten Service-Portfolio (200+ Services) und der profitabelsten Cloud-Margin. Strukturell gilt: Die meisten Fortune-500-Unternehmen haben langfristige AWS-Verträge.

Risiken aus dem Marktumfeld:

  • Azure wächst schneller (Microsoft OpenAI + Enterprise-Integration)
  • Google Cloud gewinnt mit Gemini-Modellen Terrain
  • Hoher AI-Capex ($43,2 Mrd. Q1 allein) komprimiert kurzfristig FCF und könnte Erwartungen enttäuschen
